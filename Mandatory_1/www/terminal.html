Terminal Commands for Node.js

Node

What is npm?

npm makes it easy for JavaScript developers to share and reuse code, and makes it easy to update the code that youâ€™re sharing, so you can build amazing things.

To install write the following in the terminal.

$ Install npm

Check that you have npm installed.

$ npm -v

Init

Creates a package.json file

$ npm init

Express

This is a Node.js module available through the npm registry

Before installing, download and install Node.js. Node.js 0.10 or higher is required.

$ npm install express

Nodemon

nodemon is a tool that helps develop node.js based applications by automatically restarting the node application when file changes in the directory are detected.

nodemon will be installed globally to your system path by typing the following into the terminal.

$ npm install -g nodemon

MySQL

To connect to the MySQL Database

Before you are connecting to the MySQL database, you have to have MySQL installed on your project. You are doing this by typing the command below into the terminal.

$ npm install mysql

Folder Navigation

How do you find the correct folder you want to work with?

To go through your folders in the terminal you type: 

$ cd 'the folder that comes next in the folder tree'

To go back from a folder type: 

$ ..cd

To see whats inside the current folder type: 

$ ls

Make sure you are inside the correct folder when you type in all the commands above.
